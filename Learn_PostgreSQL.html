Welcome to the PostgreSQL Tutorial

What is PostgreSQL?

  • PostgreSQL is a robust, object-relational database system that supports advanced data types and performance optimization features. It is known for its reliability, data integrity, and support for complex queries.

Getting Started with PostgreSQL

On Windows:

  • Download the installer from the PostgreSQL official website.
  • Run the installer and follow the setup wizard to install PostgreSQL.

On macOS:

  • You can use Homebrew to install PostgreSQL:
    • brew install postgresql
  • After installation, start the PostgreSQL service:
    • brew services start postgresql

On Linux:

  • Install PostgreSQL using the package manager for your distribution. For Debian-based systems:
    • sudo apt update
sudo apt install postgresql postgresql-contrib
  • Start the PostgreSQL service:
    • sudo service postgresql start

Accessing PostgreSQL

Command-Line Interface (CLI):

Open a terminal and use psql to access the PostgreSQL prompt:

psql -U postgres

- -U postgres specifies the username to connect as.

Graphical User Interface (GUI):

Tools like pgAdmin provide a graphical interface for managing PostgreSQL databases.

PostgreSQL Basic Commands and Learning Resources

Basic Commands in PostgreSQL

Connect to a Database:

\c mydatabase

List Databases:

\l

List Tables:

\dt

Describe Table Structure:

\d tablename

Exit psql:

\q

Creating and Managing Databases

Create a Database:

CREATE DATABASE mydatabase;

Drop a Database:

DROP DATABASE mydatabase;

Create a User:

CREATE USER myuser WITH PASSWORD 'mypassword';

Grant Privileges:

GRANT ALL PRIVILEGES ON DATABASE mydatabase TO myuser;

Creating Tables and Inserting Data

Create a Table:

CREATE TABLE employees (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    position VARCHAR(50),
    salary NUMERIC
);

Insert Data:

INSERT INTO employees (name, position, salary) VALUES
('John Doe', 'Developer', 60000),
('Jane Smith', 'Manager', 80000);

Select Data:

SELECT * FROM employees;

Update Data:

UPDATE employees SET salary = 65000 WHERE name = 'John Doe';

Delete Data:

DELETE FROM employees WHERE name = 'Jane Smith';

Advanced Features

Joins:

SELECT employees.name, departments.department_name
FROM employees
JOIN departments ON employees.department_id = departments.id;

Indexes:

CREATE INDEX idx_employee_name ON employees(name);

Transactions:

BEGIN;

INSERT INTO employees (name, position, salary) VALUES ('Alice Brown', 'Analyst', 55000);

COMMIT;

Views:

CREATE VIEW high_salary_employees AS
SELECT name, salary FROM employees WHERE salary > 60000;

Backing Up and Restoring Databases

Backup a Database:

pg_dump mydatabase > mydatabase_backup.sql

Restore a Database:

psql mydatabase < mydatabase_backup.sql
