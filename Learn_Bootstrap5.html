Welcome to the Bootstrap 5 Tutorial

you must have basic knowledge of HTML, CSS, before learning Bootstrap 5.

What is Bootstrap?

  • Bootstrap is a free front-end framework for faster and easier web development
  • Bootstrap includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels and many other, as well as optional JavaScript plugins
  • Bootstrap also gives you the ability to easily create responsive designs

Key Features and Changes in Bootstrap 5

  • No More jQuery: Bootstrap 5 has removed jQuery as a dependency, which makes it lighter and faster. This change also means that Bootstrap now fully leverages native JavaScript.
  • Enhanced Grid System: Bootstrap 5 introduces a new tier in the grid system, enabling more flexible and responsive designs. It includes new utilities for controlling the placement and alignment of elements.
  • Improved Forms: Forms have been completely redesigned in Bootstrap 5. They now support a more consistent look and feel across all browsers, and new custom controls have been added.
  • Utility API: The new utility API allows for easier customization of utility classes. You can now create custom utilities directly in your Sass files.
  • RTL Support: Bootstrap 5 includes built-in support for right-to-left (RTL) languages, making it easier to create websites in languages like Arabic and Hebrew.
  • Icons: While Bootstrap doesn't include its own icon library in Bootstrap 5, the official Bootstrap Icons are available as a separate project. You can easily integrate them into your projects.
  • Improved Customization: The customization process has been simplified with Bootstrap 5. You can now easily modify and extend the default Sass files to create custom themes.
  • Responsive Font Sizes: Bootstrap 5 supports responsive font sizes, which means that font sizes will automatically adjust based on the viewport size.

Setting up project

You can include Bootstrap 5 in your project using a CDN or by downloading the Bootstrap files:

Using CDN: Add the following lines to the <head> section of your HTML file: 

<!doctype html>
<html lang="en">
  <head>
    <!-- Required meta tags -->
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">

    <!-- Bootstrap CSS -->
    <link href="https://stackpath.bootstrapcdn.com/bootstrap/5.0.0/css/bootstrap.min.css" rel="stylesheet">

    <title>Hello, Bootstrap 5!</title>
  </head>
  <body>
    <h1>Hello, world!</h1>

    <!-- Optional JavaScript; choose one of the two! -->

    <!-- Option 1: Bootstrap Bundle with Popper -->
    <script src="https://stackpath.bootstrapcdn.com/bootstrap/5.0.0/js/bootstrap.bundle.min.js"></script>

  </body>
</html>

Forms

Here's a simple form using Bootstrap classes:

<!-- Form -->
<form>
  <div class="mb-3">
    <label for="exampleInputEmail1" class="form-label">Email address</label>
    <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp">
    <div id="emailHelp" class="form-text">We'll never share your email with anyone else.</div>
  </div>
  <div class="mb-3">
    <label for="exampleInputPassword1" class="form-label">Password</label>
    <input type="password" class="form-control" id="exampleInputPassword1">
  </div>
  <button type="submit" class="btn btn-primary">Submit</button>
</form>
