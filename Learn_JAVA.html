Welcome to the Java Tutorial

Learn Java from basics to advanced topics

Introduction

Java is a high-level, class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible. It is a general-purpose programming language intended to let application developers write once, run anywhere (WORA).

Java Basics

In this section, we will cover the basic concepts and syntax of Java.

  • Variables: Declare and initialize variables.
  • Data Types: Use different data types in Java.
  • Operators: Use arithmetic, relational, and logical operators.
  • Control Flow: Implement if-else statements, loops, and switch-case.
public class Main {
    public static void main(String[] args) {
        // Variables and Data Types
        int number = 10;
        double pi = 3.14;
        char letter = 'A';
        boolean isJavaFun = true;

        // Operators
        int sum = number + 5;
        boolean isEqual = (number == 10);

        // Control Flow
        if (isJavaFun) {
            System.out.println("Java is fun!");
        } else {
            System.out.println("Java is not fun.");
        }

        for (int i = 0; i < 5; i++) {
            System.out.println(i);
        }

        switch (letter) {
            case 'A':
                System.out.println("Excellent!");
                break;
            default:
                System.out.println("Grade not available.");
        }
    }
}

Classes and Objects

Java is an object-oriented programming language. In this section, we will learn how to create classes and objects.

  • Class Declaration: Define a class with fields and methods.
  • Object Creation: Create objects from a class.
  • Constructors: Initialize objects using constructors.
  • Methods: Define and call methods in a class.
class Person {
    String name;
    int age;

    // Constructor
    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Method
    void introduce() {
        System.out.println("Hello, my name is " + name + " and I am " + age + " years old.");
    }
}

public class Main {
    public static void main(String[] args) {
        // Create an object
        Person person = new Person("John", 25);

        // Call method
        person.introduce();
    }
}

Inheritance

Inheritance is a mechanism wherein a new class inherits the properties and behaviors of an existing class. In this section, we will learn about inheritance in Java.

  • Super Class: The class whose properties and behaviors are inherited.
  • Sub Class: The class that inherits the properties and behaviors of another class.
  • Method Overriding: Define a method in a subclass that has the same name and parameters as a method in the superclass.
class Animal {
    void makeSound() {
        System.out.println("Animal makes a sound");
    }
}

class Dog extends Animal {
    @Override
    void makeSound() {
        System.out.println("Dog barks");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myDog = new Dog();
        myDog.makeSound();
    }
}

Interfaces

An interface in Java is a blueprint of a class. It has static constants and abstract methods. In this section, we will learn about interfaces in Java.

  • Declare Interface: Define an interface with abstract methods.
  • Implement Interface: Implement an interface in a class.
  • Multiple Inheritance: Implement multiple interfaces in a class.
interface Animal {
    void eat();
    void sleep();
}

class Dog implements Animal {
    @Override
    public void eat() {
        System.out.println("Dog eats");
    }

    @Override
    public void sleep() {
        System.out.println("Dog sleeps");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.eat();
        myDog.sleep();
    }
}

Exception Handling

Exception handling in Java is a powerful mechanism to handle runtime errors. In this section, we will learn how to handle exceptions in Java.

  • Try-Catch: Enclose code that might throw an exception in a try block and handle exceptions in the catch block.
  • Finally: A block that will execute regardless of whether an exception is thrown or not.
  • Throw: Explicitly throw an exception.
  • Throws: Declare an exception in the method signature.
public class Main {
    public static void main(String[] args) {
        try {
            int[] numbers = {1, 2, 3};
            System.out.println(numbers[5]);
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println("Array index is out of bounds!");
        } finally {
            System.out.println("This block is always executed.");
        }
    }

    // Method that throws an exception
    public static void checkAge(int age) throws Exception {
        if (age < 18) {
            throw new Exception("Age must be 18 or above.");
        }
    }
}

Advanced Java

In this section, we will cover more advanced topics in Java, such as generics, collections, and lambda expressions.

  • Generics: Enable types (classes and interfaces) to be parameters when defining classes, interfaces, and methods.
  • Collections: Framework that provides an architecture to store and manipulate the group of objects.
  • Lambda Expressions: Provide a clear and concise way to represent one method interface using an expression.
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        // Generics
        List stringList = new ArrayList<>();
        stringList.add("Hello");
        stringList.add("World");

        for (String str : stringList) {
            System.out.println(str);
        }

        // Lambda Expressions
        stringList.forEach(s -> System.out.println(s));
    }
}
