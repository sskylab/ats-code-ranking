Welcome to the HTML Tutorial

Learn HTML from basics to advanced topics

Introduction

HTML (HyperText Markup Language) is the standard markup language for creating web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

Basic Structure of an HTML Document

An HTML document has a basic structure that includes the following elements:

  • <!DOCTYPE html>: Declares the document type and version of HTML.
  • <html>: The root element of an HTML document.
  • <head>: Contains meta-information about the HTML document.
  • <title>: Specifies the title of the document.
  • <body>: Contains the content of the HTML document.
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>

</body>
</html>

HTML Elements

HTML elements are the building blocks of HTML pages. HTML elements are represented by tags. Each element has a start tag and an end tag, with content in between.

  • Heading: <h1> to <h6> tags are used to define headings.
  • Paragraph: <p> tag is used to define a paragraph.
  • Links: <a> tag is used to define hyperlinks.
  • Images: <img> tag is used to embed images.
<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<a href="https://www.example.com">This is a link</a>
<img src="image.jpg" alt="Description of Image">

HTML Attributes

HTML attributes provide additional information about HTML elements. Attributes are always specified in the start tag and usually come in name/value pairs like name="value".

  • href: Specifies the URL of the page the link goes to.
  • src: Specifies the path to the image to be displayed.
  • alt: Provides an alternative text for the image, if the image cannot be displayed.
  • style: Used to add inline styles to an element.
<a href="https://www.example.com">This is a link</a>
<img src="image.jpg" alt="Description of Image">
<p style="color:red;">This is a paragraph.</p>

HTML Forms

HTML forms are used to collect user input. The form element itself is just a container for other elements like input elements, checkboxes, radio buttons, submit buttons, etc.

  • <form>: Defines an HTML form for user input.
  • <input>: Defines an input control.
  • <label>: Defines a label for an <input> element.
  • <button>: Defines a clickable button.
<form action="/submit-form" method="post">
    <label for="fname">First name:</label>
    <input type="text" id="fname" name="fname"><br>
    <label for="lname">Last name:</label>
    <input type="text" id="lname" name="lname"><br><br>
    <input type="submit" value="Submit">
</form>

HTML Tables

HTML tables are used to display data in a tabular format. A table consists of rows and columns. The <table> element defines the table, <tr> defines a row, <th> defines a header cell, and <td> defines a data cell. 

<table>
  <tr>
    <th>Firstname</th>
    <th>Lastname</th> 
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>Doe</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Anna</td>
    <td>Smith</td>
    <td>30</td>
  </tr>
</table>

HTML Lists

HTML lists are used to group a set of related items. There are three types of lists:

  • Ordered List: <ol> is used for ordered lists, where each item is numbered.
  • Unordered List: <ul> is used for unordered lists, where each item is bulleted.
  • Description List: <dl> is used for description lists, where each item has a description.
<h3>Ordered List</h3>
<ol>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ol>

<h3>Unordered List</h3>
<ul>
  <li>Item A</li>
  <li>Item B</li>
  <li>Item C</li>
</ul>

<h3>Description List</h3>
<dl>
  <dt>Term 1</dt>
  <dd>Definition of Term 1</dd>
  <dt>Term 2</dt>
  <dd>Definition of Term 2</dd>
</dl>

Semantic HTML

Semantic HTML is the use of HTML tags to reinforce the meaning of the information in webpages rather than merely to define its presentation. Examples of semantic elements include:

  • <header>: Represents a container for introductory content or a set of navigational links.
  • <nav>: Defines a set of navigation links.
  • <section>: Defines a section in a document.
  • <article>: Represents a self-contained piece of content that could be independently distributed or reused.
  • <footer>: Represents a footer for its nearest sectioning content or sectioning root element.
<header>
  <h1>Welcome to My Website</h1>
  <nav>
    <ul>
      <li><a href="#home">Home</a></li>
      <li><a href="#about">About</a></li>
      <li><a href="#contact">Contact</a></li>
    </ul>
  </nav>
</header>

<section>
  <article>
    <h2>Article Title</h2>
    <p>This is an article.</p>
  </article>
</section>

<footer>
  <p>Footer information goes here.</p>
</footer>
