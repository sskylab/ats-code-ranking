Welcome .NET and C# Tutorial

Introduction to .NET and C#

  • .NET Framework: A software development framework developed by Microsoft. It includes a large class library and provides support for various programming languages, including C#.

    • .NET Core / .NET 5+: A cross-platform, open-source version of .NET. The .NET ecosystem has unified into a single platform under .NET 5 and later versions (e.g., .NET 6, .NET 7, etc.).

  • C#: A modern, object-oriented programming language developed by Microsoft. It is the primary language used with the .NET framework.

Basic Syntax and Concepts in C#

Getting Start with C#

Hello World Program


using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello, World!");
    }
}

Variables and Data Types

C# supports a variety of data types. Here are some common ones:

  • int: Integer numbers
  • double: Floating-point numbers
  • char: Single characters
  • string: Textual data
  • bool: Boolean values (true or false)

int age = 25;
double height = 5.9;
char initial = 'A';
string name = "John";
bool isStudent = true;

Control Flow

Control flow statements allow you to execute different blocks of code based on conditions.

If-Else Statement


int age = 20;

if (age >= 18)
{
    Console.WriteLine("You are an adult.");
}
else
{
    Console.WriteLine("You are a minor.");
}

Switch Statement


int day = 3;

switch (day)
{
    case 1:
        Console.WriteLine("Monday");
        break;
    case 2:
        Console.WriteLine("Tuesday");
        break;
    case 3:
        Console.WriteLine("Wednesday");
        break;
    default:
        Console.WriteLine("Weekend");
        break;
}

Loops

for Loop


for (int i = 0; i < 5; i++)
{
    Console.WriteLine(i);
}

while Loop


int i = 0;
while (i < 5)
{
    Console.WriteLine(i);
    i++;
}

foreach Loop


string[] names = { "Alice", "Bob", "Charlie" };
foreach (string name in names)
{
    Console.WriteLine(name);
}

Methods

Methods are blocks of code that perform a specific task and can be called from other parts of the program.


class Program
{
    static void Main()
    {
        int result = Add(5, 3);
        Console.WriteLine(result);
    }

    static int Add(int a, int b)
    {
        return a + b;
    }
}

Classes and Objects

C# is an object-oriented language, meaning it uses classes and objects to model real-world entities.

Class Definition


public class Person
{
    public string Name { get; set; }
    public int Age { get; set; }

    public void Greet()
    {
        Console.WriteLine($"Hello, my name is {Name} and I am {Age} years old.");
    }
}

Creating and Using an Object


class Program
{
    static void Main()
    {
        Person person = new Person
        {
            Name = "John",
            Age = 30
        };
        person.Greet();
    }
}

Inheritance

Inheritance allows one class to inherit properties and methods from another.

Base Class


public class Animal
{
    public void Eat()
    {
        Console.WriteLine("Eating...");
    }
}

Derived Class


public class Dog : Animal
{
    public void Bark()
    {
        Console.WriteLine("Woof!");
    }
}

Using Inheritance


class Program
{
    static void Main()
    {
        Dog myDog = new Dog();
        myDog.Eat(); // Inherited method
        myDog.Bark(); // Derived method
    }
}

Interfaces

Interfaces define a contract that classes can implement.

Defining an Interface


public interface IAnimal
{
    void MakeSound();
}

Implementing an Interface


public class Cat : IAnimal
{
    public void MakeSound()
    {
        Console.WriteLine("Meow!");
    }
}

Exception Handling

Exception handling allows you to manage errors and exceptions gracefully.


try
{
    int result = 10 / 0;
}
catch (DivideByZeroException ex)
{
    Console.WriteLine("Cannot divide by zero.");
}
finally
{
    Console.WriteLine("This block executes regardless of whether an exception was thrown.");
}
