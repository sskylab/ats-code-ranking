Welcome to the CSS Tutorial

Learn CSS from basics to advanced topics

Introduction

CSS (Cascading Style Sheets) is a stylesheet language used for describing the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS Basics

In this section, we will cover the basic concepts and syntax of CSS.

  • Selectors: Used to select the HTML elements you want to style.
  • Properties: Used to specify what aspect of the element you want to change.
  • Values: Used to specify the value of the property.
  • Declarations: A combination of a property and a value, separated by a colon.
  • Rulesets: A combination of one or more selectors and a declaration block.
selector {
  property: value;
}

Selectors

Selectors are used to "find" (or select) the HTML elements you want to style. We will cover several types of selectors:

  • Element Selector: Selects all elements of a given type.
  • Class Selector: Selects all elements with a given class attribute.
  • ID Selector: Selects a single element with a given id attribute.
  • Attribute Selector: Selects elements based on an attribute or attribute value.
  • Pseudo-class Selector: Selects elements based on their state.
  • Pseudo-element Selector: Selects and styles a part of an element.
// Element selector
p {
  color: blue;
}

// Class selector
.intro {
  font-weight: bold;
}

// ID selector
#header {
  background-color: lightgray;
}

CSS Box Model

All HTML elements can be considered as boxes. The CSS box model is essentially a box that wraps around HTML elements, and it consists of: margins, borders, padding, and the actual content.

  • Content: The content of the box, where text and images appear.
  • Padding: Clears an area around the content; inside of the border.
  • Border: A border that goes around the padding and content.
  • Margin: Clears an area outside the border; transparent.
div {
  width: 300px;
  padding: 25px;
  border: 25px solid navy;
  margin: 25px;
}

Flexbox

Flexbox is a one-dimensional layout method for laying out items in rows or columns. It is designed to distribute space along a single row or column and provides powerful alignment capabilities.

  • Container: The parent element that holds the flex items.
  • Items: The child elements that are the flexible items within the flex container.
/* Container */
.flex-container {
  display: flex;
  flex-direction: row;
  justify-content: center;
  align-items: center;
}

/* Items */
.flex-item {
  padding: 10px;
  margin: 5px;
  background-color: lightcoral;
}

CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web. It lets you layout items in rows and columns, and it provides more control over the layout of elements on a web page.

  • Container: The parent element that holds the grid items.
  • Items: The child elements that are placed inside the grid container.
/* Container */
.grid-container {
  display: grid;
  grid-template-columns: auto auto auto;
  gap: 10px;
}

/* Items */
.grid-item {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}

Responsive Design

Responsive web design is about creating web pages that look good on all devices. It uses flexible layouts, flexible images, and media queries to achieve a responsive design.

  • Flexible Layouts: Use relative units like percentages, rather than absolute units like pixels.
  • Flexible Images: Ensure images scale nicely to fit their containing elements.
  • Media Queries: Apply different styles for different devices or screen sizes.
/* Media query for tablets */
@media (max-width: 768px) {
  .container {
    grid-template-columns: auto auto;
  }
}

/* Media query for mobile phones */
@media (max-width: 480px) {
  .container {
    grid-template-columns: auto;
  }
}
